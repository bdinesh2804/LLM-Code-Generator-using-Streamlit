# LLM Code Generator using Streamlit

A web-based application that generates programming code from natural language prompts using a Large Language Model (LLM).

## Features
- Generate code using natural language
- Supports Python, Java, C, JavaScript
- Built with Streamlit
- Uses Groq LLaMA-3.1 API

## Tech Stack
- Python
- Streamlit
- Groq API
- LLaMA-3.1 Model

## Setup Instructions
1. Clone the repository
2. Create and activate virtual environment
3. Install dependencies
4. Add API key in `.env`
5. Run Streamlit app

## Run
```bash
python -m streamlit run app.py
